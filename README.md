
<h1>Deepfake & Synthetic Media Detection Dataset</h1>
🔗 https://www.kaggle.com/datasets/payaldhokane/deepfake-and-synthetic-media-detection-dataset
<br><br>
<b>This dataset contains synthetic metadata representing real and AI-generated (deepfake) media samples across images, videos, and audio. It is designed for deepfake detection research and machine learning classification tasks.

The dataset includes simulated forensic indicators such as lip-sync consistency, visual artifact scores, compression levels, lighting inconsistencies, and generation methods. It enables researchers and practitioners to develop and evaluate deepfake detection models in a safe and ethical manner.</b>

<h3>This dataset is suitable for:</h3>

Binary Classification (Real vs Fake)

Deepfake Detection Research

Feature Importance Analysis

AI-generated Content Forensics
